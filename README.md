# Mojo CLI
This is simple CLI for ```MojoLauncher``` (mobile launcher java minecraft)

![](mojo.jpg)

# How to install

```bash
git clone https://github.com/Nos0kkk/mojo-CLI.git 

cd mojo-CLI/MOJO_DEB

pkg install ./mojo.deb

# or
dpkg -i ./mojo.deb
```

# Usage

After installation in the 
```bash
mojo --help
```
This way you will see more information
